Tài liệu Zeni Cloud — Bắt đầu trong 5 phút

Cloud SaaS toàn năng cho lập trình viên Việt Nam. Triển khai code, gọi AI, sync dữ liệu, ký smart contract — tất cả trong một dashboard duy nhất, hoá đơn VAT đỏ và tích hợp cổng nội địa.

Kiến trúc 6 lớp Zeni Cloud

Toàn bộ nền tảng được xây trên 6 lớp tiêu chuẩn — mỗi lớp dùng độc lập, kết hợp tự do qua một API token duy nhất.

L1 · COMPUTE
Cloud Run
Container serverless, tự scale 0 → 1000.
L2 · DATA
Cloud SQL + pgvector
Postgres managed, hỗ trợ vector embedding.
L3 · AI
ZeniRouter
Gemini, Anthropic, Imagen, OCR, dịch thuật.
L4 · AUTOMATION
Cron + Webhook
Tự động hoá quy trình, SMS, Slack, retry.
L5 · IDENTITY
JWT + OAuth + MFA
Đăng nhập một lần, phân quyền rõ ràng.
L6 · WEB3
Polygon
Smart contract, NFT, ZENI token.
